{% trans 'FAQ' %}

{% trans "What it TIMA?" %}

{% blocktrans %}TIMA is a database for associations. The entire project stands under the terminology "Citizen Science". There is no product behind TIMA and every information is freely available under the Creative Commons License.{% endblocktrans %}

{% trans "What are associations?" %}

{% blocktrans %}An association is the first thought that you have in mind by looking at the displayed word.{% endblocktrans %}

{% trans "What can I use TIMA for?" %}

{% blocktrans %}The data are freely available for everyone and can be used for nearly every purpose. Limitations are defined in the license.{% endblocktrans %}

{% trans "How to use TIMA?" %}

{% blocktrans %}You can use TIMA anonymously or as a registered user. After the language selection a word will be displayed. Now you can enter your association for this word. Repeat this process as long as you wish. ☺{% endblocktrans %}

{% trans "What does TIMA mean?" %}

{% blocktrans %}TIMA stands for "TIMA ist my association". This works for several languages like German "TIMA ist mein Assoziation" or Afrikaans "TIMA is my assosiasie".{% endblocktrans %}

{% trans "Why does TIMA exists?" %}

{% blocktrans %}TIMA was created during a lecture about "Citizen Science" at the University of Leipzig.{% endblocktrans %}

{% trans "Who is behind TIMA?" %}

{% blocktrans %}TIMA was created by three students and is still supported by them.{% endblocktrans %}

{% trans "What should I enter into TIMA?" %}

{% blocktrans %}Basicly you should enter the first though you have on the displayed word. This can be a noun, a verb, a adjective or multi-word unit. For example "ball", "run", "cute" or "Star Wars", ... It is important to check your spelling. TIMA tries to warn you of misspelled words as best as it can.{% endblocktrans %}

{% trans "How many languages does TIMA support?" %}

{% blocktrans %}As many as you want it to be. TIMA tries to support as many languages as possible and we attempt to add constantly more. In order to add a new language we need a small basic set of words and a way to check the spelling. If you want TIMA to support a language send us a friendly request.{% endblocktrans %}

{% trans "What about data protection and privacy?" %}

{% blocktrans %}TIMA collects as less private data as possible. TIMA does not sell any data. Private data like email are just visible to the user himself.{% endblocktrans %}
